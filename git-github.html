O que é GitHub?

O GitHub é uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que desenvolvedores colaborem em projetos de software, rastreiem alterações no código e gerenciem diferentes versões de seus projetos.

Para que serve o GitHub?

O GitHub é usado principalmente para:

  • Hospedar repositórios de código-fonte.
  • Colaborar com outros desenvolvedores.
  • Rastrear e gerenciar alterações no código.
  • Gerenciar projetos e tarefas.
  • Compartilhar código com a comunidade.

Passo a passo para criar um repositório no GitHub

  1. Faça login na sua conta do GitHub.
  2. No canto superior direito, clique no ícone + e selecione New repository.
  3. Digite um nome para o seu repositório.
  4. Adicione uma descrição opcional.
  5. Escolha a visibilidade do repositório (público ou privado).
  6. Marque a opção Initialize this repository with a README se desejar.
  7. Clique em Create repository.

Como clonar um repositório no desktop

  1. Abra o GitHub Desktop ou o terminal.
  2. No GitHub, navegue até o repositório que você deseja clonar.
  3. Clique no botão Code e copie a URL do repositório.
  4. No GitHub Desktop, clique em File > Clone repository. No terminal, use o comando git clone <URL do repositório>.
  5. Escolha o local onde deseja salvar o repositório clonado.
  6. Clique em Clone no GitHub Desktop ou pressione Enter no terminal.

O que é Git?

O Git é um sistema de controle de versão distribuído, usado para rastrear alterações no código-fonte durante o desenvolvimento de software. Ele permite que vários desenvolvedores trabalhem no mesmo projeto de forma simultânea, mantendo um histórico completo de todas as alterações feitas.

Para que serve o Git?

O Git é usado principalmente para:

  • Rastrear alterações no código-fonte.
  • Permitir a colaboração entre desenvolvedores.
  • Gerenciar diferentes versões de um projeto.
  • Facilitar a fusão de alterações feitas por diferentes desenvolvedores.

Como instalar o Git no computador

  1. Acesse o site oficial do Git: https://git-scm.com/

    .
  2. Baixe a versão adequada para o seu sistema operacional (Windows, macOS ou Linux).

  3. Execute o instalador e siga as instruções na tela.

  4. Após a instalação, abra o terminal (ou prompt de comando no Windows) e digite git --version para verificar se o Git foi instalado corretamente.

Como configurar o Git

  1. Abra o terminal (ou prompt de comando no Windows).
  2. Configure seu nome de usuário globalmente com o comando: git config --global user.name "Seu Nome".
  3. Configure seu e-mail globalmente com o comando: git config --global user.email "seuemail@example.com".
  4. Para verificar as configurações, use o comando: git config --list.

Principais comandos do Git

  • git init: Inicializa um novo repositório Git.

  • git clone <URL>: Clona um repositório existente.

  • git status: Verifica o status das alterações no repositório.

  • git add <arquivo>: Adiciona um arquivo ao índice (staging area).

  • git commit -m "mensagem": Confirma as alterações no repositório com uma mensagem descritiva.

    M: A letra M representa o estado Modified

    Isso significa que o arquivo já existia no repositório, mas que recebeu alguma modificação que ainda não foi registrada no Git.

    U: A letra U representa o estado Untracked,não rastreado.

    Isso significa que o arquivo ainda não existia no repositório e que ainda não teve seu registro (commit) feito no Git.

  • git push: Envia as alterações confirmadas para um repositório remoto.

  • git pull: Atualiza o repositório local com as alterações do repositório remoto.

  • git branch: Lista, cria ou exclui branches (ramificações).

  • git checkout <branch>: Muda para uma branch específica.

  • git merge <branch>: Mescla uma branch com a branch atual.

  • git log: Exibe o histórico de commits.
